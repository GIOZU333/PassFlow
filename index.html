Die Software „Pass Flow“ wird kostenlos zur Verfügung gestellt und dient zur Speicherung und Verschlüsselung von Passwörtern.

Pass Flow ist eine leichtgewichtige Konsolen-App, die deine Sicherheit ernst nimmt. Speichere deine Zugangsdaten lokal, verschlüsselt und absolut privat.

Jetzt Pass Flow herunterladen

user@desktop:~$ passflow --init

Initializing Pass Flow vault...

user@desktop:~$ passflow --add

Enter App Name: Instagram

Enter Password: ********

[SUCCESS] Password saved to Desktop/Instagram.txt (Encrypted)

user@desktop:~$ _

Warum Pass Flow?

Einfachheit trifft auf maximale Diskretion.

🔒

Lokale Verschlüsselung

Deine Passwörter werden mit militärischem Standard (AES-256) verschlüsselt, bevor sie auf die Festplatte geschrieben werden.

📂

Desktop-Integration

Für jede App wird automatisch ein .txt Dokument auf deinem Desktop erstellt. Sauber, ordentlich und immer griffbereit.

🌐

Offline-First

Keine Server, keine Accounts, kein Tracking. Pass Flow funktioniert zu 100% offline auf deinem eigenen Setup.

So funktioniert es

In drei einfachen Schritten zu mehr Sicherheit.

01

Eingabe über die Konsole

Starte Pass Flow direkt in deinem Windows-Terminal (CMD oder PowerShell). Gib den Namen der App und dein gewünschtes Passwort ein.

⌨️
02

Automatische Speicherung

Pass Flow legt sofort eine Datei auf deinem Desktop an. Der Dateiname entspricht deiner App (z.B. `Amazon.txt`), der Inhalt ist jedoch komplett verschlüsselt.

📝
03

Sicheres Auslesen

Willst du das Passwort wieder lesen? Nutze den intigrierten Passwort Öffner . Die App entschlüsselt die Daten und zeigt das Passwort an.

🔑

Sicherheit ohne Kompromisse

Pass Flow wurde für Windows-Nutzer entwickelt, die absolute Kontrolle über ihre Daten wollen. Dein Desktop ist dein Tresor. Wir liefern nur den Schlüssel.
